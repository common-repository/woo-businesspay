Solução de meio de pagamento online com as melhores taxas do mercado e antifraude by ClearSale.
Receba pagamentos por cartão de crédito, débito, transferência bancária e boleto usando BusinessPay.

== Description ==
A [BusinessPay](https://www.businesspay.com.br) oferece as melhores soluções em meios de pagamento online com antifraude by ClearSale usando as principais bandeiras do mercado e emissão de boletos, com as menores taxas de administração e de antecipação de recebíveis do mercado.
Gostou desse plugin? Ajude a comunidade fazendo uma avaliação ;)
Sugestões serão sempre bem-vindas.

= Para Vendedores =
Usar a BusinessPay é a melhor maneira de vender no Brasil usando o WooCommerce. Oferecemos as melhores taxas do mercado e modelo de antecipação diferenciados.
A integração é fácil e rápida, e você tem total controle das informações de pagamento e antifraude na tela de edição do pedido, ajudando a identificar rapidamente qual a situação atual do pedido.
Na tela do pedido agora oferecemos também a opção de cancelamento (estorno) no gateway de pagamento. Você não precisa mais entrar no painel de controle do gateway para realizar o estorno.
Através do painel de administração dos pedidos do WooCommerce, você tem acesso a todas as informações disponíveis sobre a transação, os status informados pelo software antifraude e rede adquirente e a captura (efetivação), para que você tenha total segurança em todas as transações.
Este plugin utiliza antifraude da ClearSale nativamente, isso significa que você não precisa se preocupar com configurações, nós cuidamos de tudo pra você.

= Os seguintes produtos antifraude da ClearSale são 100% compatíveis com este plugin: =
* Realtime Decision (resposta de aprovação em tempo real);
* ClearSale Total (grande parte da resposta em tempo real e uma parcela de análise de mesa);
* Total Garantido (mesmo modo do Total mas com garantia de recebimento do valor do pedido);
* Application (análise de mesa por sua equipe através do software ClearSale);

Saiba mais em [ClearSale para E-Commerce](https://br.clear.sale/antifraude/ecommerce).

Solicite a abertura de sua conta e começe já em [www.businesspay.com.br](https://www.businesspay.com.br).

= Para Consumidores =
Para o consumidor, oferecemos a melhor experiência de compra possível na internet, com rapidez, facilidade de uso e segurança padrão PCI Compliance.
O layout oferece cartão visual interativo na página de finalização do pedido para facilitar a inserção dos dados do cartão.

= As seguintes formas de pagamento são compatíveis: =
* Cartão de Crédito à vista ou parcelado em até 12x;
* Cartão de débito;
* Boleto Bancário via Bradesco ou Itaú;
* Transferência bancária via Bradesco ou Itaú;

= As seguintes bandeiras de cartão de crédito são compatíveis: =
* American Express;
* Aura;
* Discover;
* Elo;
* Hiper;
* Hipercard;
* JCB;
* Mastercard;
* Mastercard Maestro;
* Visa;
* Visa Electron;

= Compatibilidade =
O plugin funciona a partir da versão 5.4 do PHP, Wordpress 4.7 e WooCommerce 3.0 até suas versões mais recentes.
O plugin é compatível com visualizações em smartphones e tablets.

== Antifraude ==
Caso tenha alguma dúvida sobre o antifraude por favor entre em contato conosco [clicando aqui](https://www.businesspay.com.br/contato).

== Installation ==
= Como instalar o BusinessPay: =
* Insira os arquivos no diretório wp-contents/plugins da sua instalação do Wordpress;
* Ative o plugin;
* Efetue o cadastro no site da BusinessPay;
* Nossa equipe irá auxiliar você caso seja necessário mais algum procedimento;
* Boas Vendas :)

== Frequently Asked Questions ==
= Como faço para criar minha conta? =
* É fácil e rápido. Após cadastro no site [www.businesspay.com.br](https://www.businesspay.com.br) você receberá o contato de um especialista que irá fornecer as credenciais de acesso. Basta inserí-las na tela de configuração do plugin que já estará funcionando.

= Este plugin é desenvolvido pela BusinessPay? =
* Sim, este plugin é desenvolvido e atualizado por nossa equipe interna de tecnologia.

= Como posso falar com o suporte técnico? =
* Você pode entrar em contato com um de nossos especialistas através do site. Prometemos responder rapidamente. [www.businesspay.com.br](https://www.businesspay.com.br).

= É possível testar o plugin antes de contratar? =
* Sim, ao se cadastrar você receberá também as credenciais para o ambiente Sandbox para que você possa testar em sua plataforma antes de iniciar a operação em produção.

= O plugin funciona com o antifraude desabilitado? =
* Sim, mas os pagamentos serão capturados automaticamente, sem possibilidade de verificação manual por enquanto.

= Preciso ter uma conta na Clearsale para utilizar o antifraude? =
* Não, sua conta da BusinessPay estará integrada automaticamente.
